## Önkoşullar

* Flutter SDK (Dart dahil) kurulu olmalı.

* Web: Google Chrome.

* Android: Android Studio + Android SDK, emulator veya USB ile cihaz.

* iOS/macOS: Xcode, (iOS için) CocoaPods.

## Web’de Önizleme (en hızlı)

1. Bağımlılıkları al: `flutter pub get`
2. Çalıştır: `flutter run -d chrome`
3. Terminalde çıkan `http://localhost:xxxx` adresini tarayıcıda aç.

## Android’de Önizleme

1. Cihazları listele: `flutter devices`
2. Çalıştır: `flutter run -d <deviceId>` (emulator veya gerçek cihaz).

## iOS’ta Önizleme

1. Xcode ve simülatör hazır olmalı; gerekirse `pod install`.
2. Çalıştır: `flutter run -d ios`.

## Masaüstünde Önizleme

* macOS: `flutter run -d macos`

* Windows: `flutter run -d windows`

* Linux: `flutter run -d linux`
